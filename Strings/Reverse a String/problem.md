# Problem: Reverse a String

You are given a string `s`, and your task is to **reverse the string**.

## Examples

- **Input:** `s = "Geeks"`  
  **Output:** `"skeeG"`

- **Input:** `s = "for"`  
  **Output:** `"rof"`

- **Input:** `s = "a"`  
  **Output:** `"a"`

## Constraints

- `1 <= s.length <= 10^6`
- `s` contains only alphabetic characters (both **uppercase** and **lowercase**).

## Expected Complexities

- **Time Complexity:** `O(n)`
- **Auxiliary Space:** `O(1)`
